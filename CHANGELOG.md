# Changelog

## [0.3.0] — 2026-06-30

### Додано
- Перемикач шарів карти в експортованому HTML-файлі з позначками: ортофото (Esri, за замовчуванням), вулиці (OSM), топографічна
- Можливість скопіювати карту як зображення або зберегти у файл PNG/JPEG (кнопка в заголовку карти)
- Пункти меню «Повідомити про баг» та «Запропонувати функцію» у меню «Довідка» з переходом на GitHub Issues
- Шаблони GitHub Issues: баг-репорт (укр/англ) та запит функції (укр/англ)
- `PULL_REQUEST_TEMPLATE.md` — шаблон для pull request'ів
- `SECURITY.md` — політика відповідального розкриття вразливостей
- Динамічні посилання на файли останнього релізу на сторінці docs/index.html

### Виправлено
- Некоректне відображення кириличних символів в інсталяторі Windows (NSIS): додано прапор `/INPUTCHARSET UTF8`
- Нечитабельний текст у діалогах «Безпека даних», «Масові позначки», «Про програму» та інших на темній темі Windows — усі вбудовані HTML-діалоги тепер явно фіксують колір тексту та фон

## [0.2beta] — 2026-06-24

### Додано
- Функція масових позначок на карті: введення списку координат у будь-якому форматі, відображення на карті, збереження у вигляді HTML-файлу з інтерактивною картою Leaflet
- Таблиця MGRS-координат у збереженому HTML-файлі (опціонально)
- Розділ «Масові позначки» у документації (GitHub Pages)
- Перезапис GitHub Pages: новий дизайн сайту

### Змінено
- Версія програми оновлена до 0.2beta

## [0.1.1beta] — 2026-06-20

### Додано
- Інсталятор Windows (NSIS): встановлення у `%PROGRAMFILES64%`, ярлики у меню Пуск, запис у «Програми та компоненти», деінсталятор; автоматичне завантаження та встановлення Visual C++ Redistributable 2022 за відсутності
- Тести для Windows, Linux (DEB, RPM, AppImage, tar.xz): встановлення, CLI-перевірка `--convert`, GUI smoke-тест через Xvfb

### Виправлено
- Конфлікт RPM-пакета з системним пакетом `filesystem` (шлях `/usr/sbin`)
- Некоректна назва іконки AppImage в CI
- Збірка для Windows та Linux в CI: правильна архітектура Qt та версія Ubuntu

## [0.1beta] — 2026-06-18

### Додано
- Конвертація координат між форматами: MGRS, DD, DMS, UTM, УСК-2000
- Інтерактивна карта на базі Leaflet (OpenStreetMap, Esri Orthoimage, OpenTopoMap)
- Відображення точки та зони MGRS на карті
- Системний трей з можливістю введення координат з буфера обміну
- Веб-сервер для інтеграції із зовнішніми застосунками (вимкнений за замовчуванням)
- Консольний режим: `--convert`, `--url`, `--open`, `--version`
- Підтримка двох мов інтерфейсу: українська та англійська
- Темна та світла тема (слідує за системною)
- CI/CD: збірка та пакування для macOS (DMG), Windows (Setup + ZIP), Linux (AppImage, DEB, RPM, tar.xz)
- Ліцензія MIT
